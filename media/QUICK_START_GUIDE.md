## Common Questions While Downloading

### Q: "I don't like the first image, can I download another?"
**A:** Yes! The guides suggest 5 images per category. Pick the ones you like best.

### Q: "Do I need to name files exactly as the guide says?"
**A:** No, but following the naming convention keeps things organized.

### Q: "Can I mix images from different platforms?"
**A:** Absolutely! Mix and match freely.

### Q: "This image doesn't download - now what?"
**A:** Try another image from the list. The guide provides multiple options.

### Q: "Do I need attribution/credits?"
**A:** Most free sources don't require it, but it's nice to give credit!

### Q: "Can I use images from other sources?"
**A:** Yes, just make sure they're free to use.

## ‚ö° Pro Tips for Speed

### Batch Download:
- Open 2-3 browser tabs
- Download multiple images at once
- Faster than going one-by-one

### Keyboard Shortcuts:
- Use Ctrl+T (or Cmd+T) to open new tabs
- Use Ctrl+W to close tabs after download

### Save Time:
- Start with Freepik (best for playful)
- Use DrawKit's collections (ready-made sets)
- Bookmark good photographers/artists

### Test As You Go:
- After downloading 3-4 images, preview in your document
- Make sure style is consistent
- Adjust if needed before downloading more
