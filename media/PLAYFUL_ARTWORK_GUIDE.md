## Step-by-Step Download Instructions

### For Freepik.com (Best cartoon selection):
1. Go to freepik.com
2. Search using keywords from above (e.g., "children talking illustration")
3. **IMPORTANT:** Filter by "Free" (checkbox on left side)
4. Click on image
5. Click "Download"
6. Choose free format (usually PNG)
7. Save to appropriate media subfolder
8. **Note attribution if required** on free content

### For DrawKit.io:
1. Go to drawkit.io
2. Browse "Illustrations" section
3. Search for relevant keywords
4. Click illustration
5. Click "Download" (SVG or PNG)
6. Save to appropriate folder
7. No attribution required

### For Undraw.co:
1. Go to undraw.co
2. Search for keyword (e.g., "communication," "helping," "family")
3. **Customize color:** Click illustration, choose color palette
4. Click "Download SVG" or "Download PNG"
5. Save to appropriate folder
6. No attribution required

### For OpenPeeps.com:
1. Go to openpeeps.com
2. Browse character illustrations
3. Mix and match to create custom scenes
4. Download PNG
5. Save to appropriate folder
6. No attribution required

### For Pixabay.com (for illustrations):
1. Go to pixabay.com
2. Search keyword
3. Filter by "Illustration"
4. Click image
5. Click "Free Download"
6. Save to appropriate folder
7. No attribution required

## Tips for Best Results

✅ **Do:**
- Download PNG files with transparent backgrounds for flexibility
- Test different sources to find your preferred illustration style
- Mix and match from different sources for variety
- Resize images appropriately for your handout
- Create a visual flow that appeals to children

❌ **Don't:**
- Use overly cartoonish images that seem unprofessional
- Mix too many different illustration styles
- Ignore licensing requirements (even for "free" content)
- Download low-resolution images (aim for 1920x1080px minimum)
